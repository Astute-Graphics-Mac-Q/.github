## 📋 Overview

<a href="#astuteGraphics">Astute Graphics</a> is the leading Adobe Illustrator plugin suite for Mac — a collection of specialized vector design plugins that extend Illustrator with precision tools, effects, and capabilities that the base application doesn't provide, serving professional illustrators, logo designers, type designers, and technical illustrators who push Illustrator's native tools to their limits.

The <a href="#astuteGraphics">VectorScribe plugin</a> is Astute Graphics' flagship — providing the Dynamic Corners tool that applies live, adjustable rounded corners to any path corner without expanding the effect, the Smart Remove Brush that removes anchor points intelligently while maintaining path shape, and path measurement tools that display the precise dimensions of any path segment directly in the canvas. These additions address the most common frustrations with Illustrator's native path handling: corners that require manual adjustment, anchor point density that builds up through operations, and the lack of direct measurement feedback during drawing. The <a href="#astuteGraphics">Phantasm color adjustment plugin</a> applies Photoshop-style color adjustments to Illustrator artwork as live effects — Hue/Saturation, Brightness/Contrast, Curves, and Color Balance adjustments that remain editable through the Appearance panel rather than destructively altering fill colors. These live color effects apply to individual objects, groups, or entire layers and remain adjustable throughout the design process.

The <a href="#astuteGraphics">WidthScribe plugin</a> provides advanced width profile editing for Illustrator's variable-width strokes — painting width variations along paths, applying gradient width profiles, and creating the organic stroke variation that calligraphic and hand-lettering styles require. Width profiles apply non-destructively and animate interactively as they are edited. The <a href="#astuteGraphics">MirrorMe plugin</a> adds live symmetry drawing to Illustrator — drawing on one side of a mirror axis and seeing the symmetrical reflection appear simultaneously, ideal for logo design, mandala creation, symmetrical illustration, and any design that benefits from perfect bilateral symmetry during creation.

<a href="#astuteGraphics">ColliderScribe plugin</a> manages the precise spatial relationships between objects — snapping, aligning, and positioning objects with exact gap control and collision detection that Illustrator's standard snapping can't provide for the tight spatial arrangements professional layout and logo design requires. <a href="#astuteGraphics">Stipplism plugin</a> converts Illustrator artwork to stipple (dot pattern) effects — creating the textured, hand-crafted stipple illustration style that editorial illustration and decorative design uses, with configurable dot density, size, and distribution.

<a href="#astuteGraphics">SubScribe plugin</a> adds construction geometry tools — drawing tangent lines and circles, bisecting angles, and creating geometric constructions that technical illustration and precise ornamental design requires without manual calculation. <a href="#astuteGraphics">Subscription-based access</a> through Astute Graphics' subscription provides access to the complete plugin suite with updates — individual plugins also available for designers who need specific tools rather than the complete collection.

## ❓ FAQ

<details>
<summary>Do Astute Graphics plugins work with the current version of Adobe Illustrator?</summary>

Yes. Astute Graphics provides compatibility updates with each major Illustrator release. Subscription users receive these updates automatically.

</details>

<details>
<summary>What is the VectorScribe Dynamic Corners tool?</summary>

Dynamic Corners applies live, adjustable rounded corners to any path corner in Illustrator. The corners remain editable and non-destructive — adjusting the radius at any time without re-drawing the path.

</details>

<details>
<summary>What does the Phantasm plugin do?</summary>

Phantasm applies Photoshop-style color adjustments (Hue/Saturation, Curves, Brightness/Contrast, Color Balance) as live, non-destructive effects on Illustrator artwork through the Appearance panel.

</details>

<details>
<summary>Does MirrorMe work for radial symmetry as well as bilateral?</summary>

Yes. MirrorMe supports bilateral and radial symmetry configurations for creating mandalas and other radially symmetric designs.

</details>

<details>
<summary>Is there a trial for Astute Graphics plugins?</summary>

Yes. Trials are available for evaluation in the actual Illustrator workflow before purchasing.

</details>

<details>
<summary>Are Astute Graphics plugins available individually or only as a suite?</summary>

Both. The complete plugin suite is available through a subscription, and individual plugins are available for separate purchase for designers who need specific tools.

</details>

<details>
<summary>Does WidthScribe work with Illustrator's variable width strokes?</summary>

Yes. WidthScribe extends Illustrator's variable width stroke capability with interactive painting of width variations along paths.

</details>

<details>
<summary>What is the InkScribe plugin?</summary>

InkScribe is an alternative drawing tool for Illustrator that replaces the Pen tool with improved curve preview during drawing and more intuitive anchor point manipulation.

</details>

<details>
<summary>Does Stipplism work with any Illustrator artwork?</summary>

Yes. Stipplism converts Illustrator vector artwork to stipple dot effects with configurable density, size, and distribution for editorial and decorative illustration styles.

</details>
